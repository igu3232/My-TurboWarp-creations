# コマンドラインもどき 関数一覧 (v0.1.0)

## 基本

### help
`help (関数名)`
・helpだけを実行すると、定義されている全ての関数の名前と一行の説明を表示します。
・helpの後に任意の関数の名前を入力すると、入力した関数の一行の説明を表示します。

### exit
`exit`
・プログラムを即座に終了します。

### clear
`clear`
・今までのコンソールに表示されていた内容をクリアします

### echo
`echo (1行目) (2行目) ・・・`
・引数に入力した内容をそのまま返します。
・複数の引数に入力すると、一つの引数を一行として引数1から順に出力します

### date
`date`
・関数を実行した時の西暦と日付をYYYY:MM:DDの形式で出力します

### time
`time`
・関数を実行した時の時間、分、秒をHH:mm:ssの形式で出力します


## 計算

### rand
`rand (下限) (上限)`
・下限から上限までの範囲で乱数を生成して出力します

### add
`add (数値1) (数値2) (保存スロット)`
・数値1と数値2を足した数値を出力します
・引数3に`slot1`のように保存するメモリースロットのスロット番号を入力すると、指定したスロットに計算結果の数値を書き込みます
・数値1と2には代わりに`slot2`のようにメモリースロットのスロット番号を指定すると、指定したスロットに書き込まれている数値を使用できます

### reduce
`reduce (数値1) (数値2) (保存スロット)`
・数値1と数値2で減算した数値を出力します
・引数3に`slot1`のように保存するメモリースロットのスロット番号を入力すると、指定したスロットに計算結果の数値を書き込みます
・数値1と2には代わりに`slot2`のようにメモリースロットのスロット番号を指定すると、指定したスロットに書き込まれている数値を使用できます

### mul
`mul (数値1) (数値2) (保存スロット)`
・数値1と数値2を掛けた数値を出力します
・引数3に`slot1`のように保存するメモリースロットのスロット番号を入力すると、指定したスロットに計算結果の数値を書き込みます
・数値1と2には代わりに`slot2`のようにメモリースロットのスロット番号を指定すると、指定したスロットに書き込まれている数値を使用できます

### div
`div (数値1) (数値2) (保存スロット)`
・数値1を数値2で割った数値を出力します
・引数3に`slot1`のように保存するメモリースロットのスロット番号を入力すると、指定したスロットに計算結果の数値を書き込みます
・数値1と2には代わりに`slot2`のようにメモリースロットのスロット番号を指定すると、指定したスロットに書き込まれている数値を使用できます

## メモリースロット

### ms
`ms`
・関数を実行した時のメモリースロットの内容を出力します

### memory
`memory (操作内容) (目標1) (目標2)`
・操作内容にclearを入力すると、全てのスロットをunusedにリセットします
・操作内容にsetを入力すると、目標1で指定したスロット番号のスロットに目標2で指定した内容を書き込みます
・操作内容にremoveを入力すると、目標1で指定したスロット番号のスロットをunusedにします
・操作内容にswapを入力すると、目標1で指定したスロット番号のスロットと目標2で指定したスロット番号のスロットを入れ替えます

## おまけ
### guess
・指定した範囲でランダムに決められた数値(お題)に対して、入力した数値がお題より大きいか小さいかのヒントを頼りにお題を探し当てるゲームです。
・guess setting (上限値)でお題が出る範囲を1～指定した数値までに設定することができます
・guess startでゲームを開始します
・guess (任意の値)で入力をして、指定した数値とお題の大小のヒントを得ます
・guess spoilersでゲームを終了して正解を出力します(諦めたとき用)
・guess endでゲームを終了します
